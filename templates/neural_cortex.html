Interactive Weight Explorer Guide
1. Synaptic Weight Connection (The Matrix Grid)

A weight value controls how strongly signal flows from a preceding input/neuron (column) to a receiving neuron (row).
• Rose / Red (+): **Positive / Bullish weight** - amplifies the signal, pushing decisions towards buying or longing assets.
• Blue (-): **Negative / Bearish weight** - inhibits the signal, pushing decisions towards selling or shorting assets.
• Dark / Gray (0.0): **Neutral weight** - connection is inactive or weak, meaning it doesn't affect decisions.

2. Strongest Bullish (+) & Bearish (-) Leaderboard

Identifies the pathways that exert the strongest influence on trading actions:
• **Strongest Bullish (+)**: The strongest pathways contributing to **buy triggers**.
• **Strongest Bearish (-)**: The strongest pathways contributing to **sell triggers**.
By looking at the leaderboard, you can immediately identify which specific coordinates determine the model's primary market stances.

3. Dimensions & Shape Coordinates

The shape represents parameters array dimension `[Rows, Cols]`.
• **Row index (Target)**: Represents the receiving neuron in the next layer.
• **Column index (Source)**: Represents the sending input feature or preceding neuron.
For example, for Input layer weights shape `[64, 4]`, column `1` represents the Sentiment input, and row `12` represents the 12th pattern detector neuron receiving it.

4. Deltas & Parameter Tuning

**Delta** shows how much the weight value has moved from the previous checkpoint during online learning.
To tune a weight manually: Enter coordinate indices in the **Target Coordinate Finder**, click **Find & Load Weight**, click **Manually Adjust Parameter**, input a value and a mandatory change reason, and confirm.

PPO Learning & Weights Guide

1. Neural Inputs (Sensory Core)

The model receives 4 inputs representing the state of the market: **Price Trend** (momentum), **Sentiment Score** (news/sentiment extraction), **Volatility** (price swing range), and **Volume/Spread** (liquidity pressure).

2. Hidden Layers (Feature Extraction)

Neurons in middle layers (Hidden 1 & 2) act as pattern detectors. They don't see the outside world directly. Instead, they extract complex triggers, like "market capitulation" or "stable consolidation," by combining inputs.

3. Action Output (Actor/Critic)

The **Actor** outputs the action signal strictly between **-1.0 (Sell)** and **+1.0 (Buy)**. The **Critic** outputs the predicted trade return to evaluate if decisions were optimal, guiding online PPO reinforcement updates.
