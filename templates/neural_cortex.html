How to read the "Brick Wall" (Synaptic Heatmap)

Each cell in the grid represents a single **synaptic weight connection** between a source neuron (x-axis / column) and a target neuron (y-axis / row).

  • Rose / Red cells indicate **Positive weight**: The connection amplifies the signal, pushing the output positive (more bullish).
  • Blue cells indicate **Negative weight**: The connection inhibits the signal, pushing the output negative (more bearish).
  • Dark / Gray cells represent **Neutral weight (near 0)**: The connection is weak and has little impact on decisions.

PPO Learning & Weights Guide

1. Neural Weights

Weights represent the strength of connections between neurons. During evaluation, input data (e.g. price trends, volume) is multiplied by weights to decide whether to Buy, Hold, or Sell. An actor weight of +0.5 means the model strongly favors action triggers for this input state.

2. Actor vs Critic

The Actor module generates action signals strictly between -1.0 (Sell) and +1.0 (Buy). The Critic module computes the value estimate (estimated future net return). The difference between the Critic's estimate and the actual trade return guides learning.

3. Online Updates

Every 8 completed trades, the model triggers a PPO micro-update. If a trade made profit net of fees, the model increases weights matching that state-action. If it lost money, weights are penalized. The reinforcement update uses a conservative learning rate (5e-5) to adjust strategy.
